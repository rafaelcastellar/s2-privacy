Política de Privacidade – S2

Português

Última atualização: 19 de junho de 2025

O S2 é um aplicativo que envia mensagens positivas e personalizadas com base no estado emocional informado pelo próprio usuário. Este app foi criado com o objetivo de promover bem-estar emocional de forma respeitosa, segura e anônima.

1. Dados coletados

O S2 não coleta dados pessoais identificáveis. Os dados armazenados se limitam a:

  • Estado emocional selecionado pelo usuário (ex: triste, alegre, ansioso);
  • Preferências locais do app (como idioma, frequência de mensagens);
  • Feedback simples (“sim” ou “não”) em relação às mensagens recebidas.

2. Armazenamento e uso dos dados

Os dados são armazenados de forma segura no Firebase Firestore e são utilizados apenas para:

  • Personalizar as mensagens enviadas;
  • Melhorar a experiência geral do app;
  • Monitorar o uso de forma agregada e anônima.

3. Compartilhamento de dados

O S2 não compartilha nenhum dado com terceiros.

4. Segurança

Todos os dados são armazenados com segurança e o app não coleta nome, e-mail, senha ou qualquer outro dado pessoal.

5. Consentimento

Ao usar o S2, você concorda com esta política de privacidade.

Este aplicativo é voltado a usuários com idade igual ou superior a 16 anos. Não é recomendado para menores dessa faixa etária.

6. Atualizações

Esta política pode ser atualizada periodicamente. Mudanças serão publicadas nesta página.

7. Contato

Em caso de dúvidas ou sugestões, entre em contato com: rafael.castellar@gmail.com

English

Last updated: June 19, 2025

S2 is an app that delivers positive and personalized messages based on the user’s reported emotional state. It was created to promote emotional well-being in a respectful, safe, and anonymous way.

1. Data collected

S2 does not collect any personally identifiable information. The data stored is limited to:

  • The emotional state selected by the user (e.g., sad, happy, anxious);
  • Local app preferences (such as language and message frequency);
  • Simple feedback ("yes" or "no") about the messages received.

2. Data storage and usage

All data is securely stored using Firebase Firestore and used solely to:

  • Personalize the messages sent to users;
  • Improve the overall app experience;
  • Monitor general usage in an anonymous and aggregated manner.

3. Data sharing

S2 does not share any data with third parties.

4. Security

All data is securely stored. The app does not collect names, emails, passwords, or any other personal information.

5. Consent

By using S2, you agree to this privacy policy.

This app is intended for users aged 16 and older. It is not recommended for users under this age.

6. Updates

This policy may be updated periodically. Changes will be published on this page.

7. Contact
